一组开源的有怀旧游戏感觉的像素字体。字体基本上以英文、数字和符号为主。

如果您在寻找开源的泛中日韩或 Unicode 像素字体，请尝试 [「方舟像素字体」](https://github.com/TakWolf/ark-pixel-font)。

这个项目不仅提供了全部的字形源文件，也提供了构建字体所需要的完整程序。
